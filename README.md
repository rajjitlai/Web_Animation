# Web Animation - Beautiful CSS Effects

A visually stunning web experience featuring fluid, interactive CSS animations. This project demonstrates the power of modern CSS3 for creating dynamic, eye-catching elements without the need for heavy JavaScript libraries.

## ✨ Features

- **Dynamic Box Animations**: Smooth scaling and rotating square elements that create a sense of depth and motion.
- **Orbital Circle Effects**: Floating circular elements that transition from squares to circles while rising, adding a layered animation feel.
- **Responsive Design**: Fully optimized for various screen sizes, from high-resolution monitors to mobile devices.
- **Glassmorphism Background**: A subtle, elegant background with radial gradients for a modern, high-end look.

## 🚀 Getting Started

### Prerequisites
To run this project, you only need a modern web browser (such as Chrome, Firefox, or Edge).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rajjitlai/Web_Animation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Web_Animation
   ```
3. Open `index.html` in your browser.
